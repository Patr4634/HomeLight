You need Expo CLI to run the development environment
npm install -g expo-cli

You may, or may not need performe  a node module install.
npm install

To start the app use,
expo start

If you get unauthorized access, then change your execution policy to unrestricted.
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Unrestricted

After you are done you can set it back if you want to
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Restricted
